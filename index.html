Menú de Métodos de Integración Numérica

La integración en métodos numéricos es una técnica para aproximar el área bajo una curva cuando no es posible o no es práctico usar una fórmula exacta. Se utiliza para estimar cantidades como distancias, volúmenes, ingresos, etc., a partir de datos discretos. Se aplica con reglas como: Regla del rectángulo Regla del trapecio Regla de Simpson 1/3 y 3/8 Cada método tiene distinta precisión y complejidad

Los valores colocados por defecto son aleatorios, coloque los datos que desee calcular en a - b - funcion - la funcion puede ser expresado en velocidadd, dinero ganado, etc
